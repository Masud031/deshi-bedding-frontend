# Deshi Bedding Frontend

Modern e-commerce frontend for Deshi Bedding.

## Tech Stack

- React 19
- TypeScript
- Vite
- TanStack Router
- Redux Toolkit
- RTK Query
- Tailwind CSS
- SweetAlert2

## Features

- Home Page
- Shop
- Product Details
- Shopping Cart
- Checkout
- Authentication
- Admin Dashboard
- Responsive Design

## Installation

```bash
npm install
npm run dev
